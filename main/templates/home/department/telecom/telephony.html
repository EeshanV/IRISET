Description:

This lab is equipped with different types of Circuit Switching basedTelephonySystems(ISDN-PBX) to impart training onhardware, software configuration and inter exchange connectivity using PRI, E1 and COinterfaces.

This lab is also equipped with Packet Switching based IP-Telephony Systems (Open source server based IP-PBX)to impart training onhardware, software configuration and inter exchange connectivity using IP- Trunks, SIP trunks, PRI Gateways and CO gateways.

Apart from these, training on IP-DSLAM Equipment and its end point ADSL Modem configuration for extending broadband and voice services to subscribers is imparted.

Equipment available in the lab:

  • ISDN based Tadiran’s- CoralFlexicom6000
  • ISDN based Siemens Hipath 3800 with WLL(Wireless in Local Loop) services
  • Asterisk Open source VoIP server based PBX-10 nos
  • IP-DSLAM Equipment and ADSL-CPE modem configuration.
