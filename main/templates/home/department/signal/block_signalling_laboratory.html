Description:
Block Signalling Lab has various types of block instruments being used on Indian Railways. Both Table top miniature models, actual Instruments & Various parts of the equipments give clear idea including inaccessible parts. Digital Flexi charts depict detailed circuits of these Block Instruments and related accessories.

This lab has the following equipments:

  • Single Line
    • >> Token Instruments - Neale's Ball Token instrument
    • >> Tokenless Instruments
      •     i. Push button type
      •     ii. Frequency Modulation (Handle) Type: Daido Type

  • Double line block instruments of makes:
    • Byculla Work shop (Central Railway)
    • Podanur work shop (Southern Railway)

  • Block Working through Axle Counters :
    • UFSBI with panel for S/L (Deltron make)
    • UFSBI with Token Block Instrument (Deltron make)
    • SSBPAC S/L (Efftronics make)
    • SSBPAC D/L (Efftronics make)
    • Axle Counter Block Working (CEL make)

  • Intermediate block signalling with Auto TOL feature.
  • Automatic Block Signalling.
